Typography

I will continue to use the same fonts that I did for my homepage which include the google api font
family Montserrat for all p tags and a tags. All h tags will be the google api font family Roboto
Condensed. Below I included the link tag to import them and how it may look in a css stylesheet as
well.

Example link tag:

link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@300&family=Robo
to+Condensed:wght@700&display=swap" rel="stylesheet"

Example css stylesheet reference:

p{
font-family: 'Montserrat', sans-serif;
}
header{
font-family: 'Roboto Condensed', sans-serif;
}
